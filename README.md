# TimeAPI SPA

## Navodila za namestitev

### Docker

V korenski mapi projekta poženi:

    docker build -t spica-app .
    docker run -p 3000:3000 -d spica-app

Aplikacija bo na voljo na http://localhost:3000/.

### Nodejs

V korenski mapi projekta poženi

    npm install
    npm start

Aplikacija bo na voljo na http://localhost:3000/.
